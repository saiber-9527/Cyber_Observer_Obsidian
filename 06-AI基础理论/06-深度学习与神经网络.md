# 深度学习与神经网络

> 用多层神经元模拟复杂函数——AI 能力爆发的核心技术。

---

## 神经元模型

```
输入 ──→ 加权求和 ──→ 激活函数 ──→ 输出
  x1 ─→ w1
          ╲
  x2 ─→ w2 ──→ Σ ──→ σ(·) ──→ y
          ╱
  x3 ─→ w3
```

- 每个输入有**权重**（w）
- 加权求和后经过**激活函数**（sigmoid / ReLU / tanh）
- 输出传给下一层

---

## 常见的网络结构

### 1. 全连接网络（MLP）
- 最简单，每层所有神经元都相连
- 用于简单分类/回归

### 2. 卷积神经网络（CNN）
- 擅长处理图像等网格结构数据
- 关键操作：卷积（特征提取）+ 池化（降维）+ 全连接（分类）

### 3. 循环神经网络（RNN/LSTM）
- 擅长序列数据（文本、时间序列）
- 有"记忆"——当前输出依赖之前的输入
- LSTM 解决了长期依赖问题（比朴素 RNN 记得更远）

### 4. Transformer（当前重点）
- 全注意力机制，没有循环结构
- 并行计算，适合大规模训练
- 详细见 [[06-Transformer架构]]

## 训练的核心：反向传播

```
正向传播：输入 → 计算 → 输出 → 计算损失

反向传播：损失 → 逐层计算梯度 → 更新权重

重复以上过程，损失越来越小
```

### 关键组件
- **优化器**：SGD、Adam、AdamW（决定如何更新权重）
- **学习率**：每次更新的步长（太大不收敛，太小收敛慢）
- **Batch Size**：每次更新用的样本数量
- **Epoch**：整个训练集遍历一遍

## 算力成本（为什么 GPU 重要）

| 模型 | 参数规模 | 训练需要的算力 |
|------|---------|-------------|
| AlexNet (2012) | 60M | 2 天 × 2 GPU |
| ResNet-152 (2015) | 60M | 数周 × 8 GPU |
| GPT-3 (2020) | 175B | 数千 GPU 训练数月 |
| LLaMA-3 (2024) | 405B | 数百万 GPU 小时 |

> 大模型的训练成本已经高到只有少数公司玩得起。
> 普通人更多的是用预训练好的模型做微调或调用 API。
