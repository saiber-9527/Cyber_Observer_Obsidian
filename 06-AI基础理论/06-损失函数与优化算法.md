# 损失函数与优化算法

> 损失函数告诉模型"你错得多离谱"，优化算法告诉它"怎么改"。

---

## 常见损失函数

### 分类任务

| 损失函数 | 公式 | 适用 |
|---------|------|------|
| **交叉熵损失 (Cross-Entropy)** | `-∑y·log(ŷ)` | 多分类（最常用） |
| **二元交叉熵 (BCE)** | `-[y·log(p) + (1-y)·log(1-p)]` | 二分类 |
| **Hinge Loss** | `max(0, 1 - y·ŷ)` | SVM |
| **Focal Loss** | `-(1-p)ᵞ·log(p)` | 类别不平衡 |

### 回归任务

| 损失函数 | 公式 | 特点 |
|---------|------|------|
| **MSE (L2 Loss)** | `(y-ŷ)²` | 对大误差惩罚大（对异常值敏感） |
| **MAE (L1 Loss)** | `|y-ŷ|` | 对异常值更鲁棒 |
| **Huber Loss** | 结合 MSE+MAE | 两者折中 |

### 安全场景特殊损失

```
对比学习的 Contrastive Loss:
  L = y·D² + (1-y)·max(0, margin-D)²
  让同类样本靠近，异类样本远离
  适用于异常检测（正常样本聚在一起，异常样本被推远）
```

## 常见优化算法

| 算法 | 特点 | 适用场景 |
|------|------|---------|
| **SGD** | 最朴素，每个 batch 更新一次 | ⚠️ 收敛慢，需要调学习率 |
| **Momentum** | SGD + 惯性（积累历史梯度） | ✅ 比 SGD 快 |
| **Adagrad** | 自适应学习率（稀疏梯度友好） | 稀疏数据 |
| **RMSProp** | 修复 Adagrad 学习率递减问题 | RNN / 非平稳目标 |
| **Adam** | Momentum + RMSProp 合体 | ✅ 最常用，优先尝试 |
| **AdamW** | Adam + 解耦权重衰减 | ✅ 推荐（比 Adam 更好） |

**经验法则：** 不确定选哪个的时候，先试 AdamW。

## 学习率调度

| 策略 | 说明 |
|------|------|
| **固定** | 整个训练周期学习率不变 |
| **Step Decay** | 每 N 个 epoch 减半 |
| **余弦退火 (Cosine Annealing)** | 学习率按余弦曲线下降 |
| **Warmup** | 前期从小学习率逐渐升到目标值 |
| **ReduceLROnPlateau** | 验证集指标停止提升时降低学习率 |

## 梯度消失 vs 梯度爆炸

```
梯度消失：深层网络的前面几层梯度接近 0，参数几乎不更新
  ── 原因：sigmoid/tanh 使梯度越来越小
  ── 解法：ReLU 激活函数、残差连接 (ResNet)、LayerNorm

梯度爆炸：梯度值过大，参数更新幅度巨大
  ── 原因：网络太深或权重初始值太大
  ── 解法：梯度裁剪 (Gradient Clipping)、权重初始化
```

## 训练小技巧

| 技巧 | 说明 |
|------|------|
| **Batch Normalization** | 每层输入标准化，加速训练 |
| **Dropout** | 训练时随机"丢弃"部分神经元，防过拟合 |
| **Early Stopping** | 验证集不再改善时提前停止训练 |
| **Weight Decay** | 权重正则化 L2 惩罚 |
| **Gradient Accumulation** | 小 batch 积累梯度后统一更新，模拟大 batch |
| **混合精度训练** | FP16 训练，显存减半速度翻倍 |
