## 学习路径

```
协议安全 ──→ 边界防御 ──→ Web 安全 ──→ 渗透测试
   │           │              │
   ├ TCP/IP    ├ 防火墙        ├ XSS
   ├ DNS       ├ IDS/IPS       ├ SQL注入
   ├ TLS/HTTPS ├ WAF           ├ CSRF
   └ HTTP/2    └ DDoS防护      └ SSRF
                │
                └ VPN → 零信任
```
