# 常见网络攻击类型

> 知己知彼——这是安全入门的"敌情地图"。

---

## 按攻击目标分类

### 1. 凭证攻击（Credential Attacks）
| 攻击 | 说明 | 典型场景 |
|------|------|---------|
| **暴力破解** | 不断尝试用户名密码组合 | SSH/Web登录爆破 |
| **凭证填充** | 用泄露的账号密码去撞其他网站 | 复用密码的后果 |
| **中间人攻击** | 拦截通信获取凭证 | 公共WiFi上登录 |
| **钓鱼** | 伪造可信页面骗取凭证 | 假冒银行/公司登录页 |

**防御：** MFA、密码管理器、证书验证、安全意识培训

### 2. Web 应用攻击
| 攻击 | 说明 | 防御 |
|------|------|------|
| **SQL 注入** | 注入 SQL 语句操纵数据库 | 参数化查询 |
| **XSS** | 注入客户端脚本 | 输出编码 + CSP |
| **CSRF** | 跨站请求伪造 | CSRF Token + SameSite |
| **SSRF** | 服务端请求伪造 | URL 白名单 |
| **文件上传漏洞** | 上传恶意文件 | 文件类型校验 + 隔离存储 |
| **命令注入** | 注入系统命令 | 输入过滤 + 最小权限 |

### 3. 拒绝服务攻击
| 攻击 | 说明 |
|------|------|
| **带宽耗尽** | SYN Flood、UDP Flood |
| **应用层攻击** | HTTP Flood、Slowloris |
| **分布式放大** | NTP/DNS 反射放大 |
| **资源耗尽** | 数据库连接耗尽、API 限流绕过 |

### 4. 网络嗅探与欺骗
| 攻击 | 说明 |
|------|------|
| **ARP 欺骗** | 伪造 ARP 报文冒充网关 |
| **DNS 劫持** | 伪造 DNS 响应 |
| **DHCP 欺骗** | 伪造 DHCP 服务器 |
| **端口扫描** | 探测开放端口和服务 |

```
         payload  钓鱼   漏洞   后门  信道
```

## 按攻击者类型分类

| 攻击者 | 目的 | 能力 |
|--------|------|------|
| **脚本小子** | 出名/好玩 | 低（用现成工具） |
| **黑客组织** | 经济利益 | 中-高 |
| **APT（高级持续威胁）** | 情报窃取/破坏 | 极高（国家级） |
| **内部威胁** | 报复/利益 | 高（有合法访问权限） |
| **勒索团伙** | 金钱 | 中-高 |
| **网络犯罪团伙** | 经济 | 中 |

## 近五年攻击趋势

| 年份 | 主要趋势 |
|------|---------|
| 2022 | 勒索软件即服务（RaaS）爆发 |
| 2023 | 供应链攻击频发（SolarWinds 之后） |
| 2024 | AI 辅助攻击兴起（自动化钓鱼/Deepfake） |
| 2025 | LLM 攻击面扩大（提示注入/模型窃取） |
| 2026 | AI Agent 攻击链（自主攻击工具调用） |

---

## ⚠️ 认知要点

1. **攻击者也在用 AI** — 不要以为只有防御者在用 AI
2. **90% 的攻击利用的是已知漏洞** — 补丁管理比"发现0day"更有效
3. **攻击者追求的是回报率** — 不是非要用最高级的手法，有用就行
4. **人是最大的漏洞** — 钓鱼和社会工程学仍然是最高效的攻击方式
