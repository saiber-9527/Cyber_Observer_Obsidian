# 关键信息基础设施（关基）保护

> 一旦出问题就会严重危害国家安全、国计民生、公共利益的信息设施。

---

## 关基认定范围

根据《关键信息基础设施安全保护条例》（2021）：

| 领域 | 典型设施 |
|------|---------|
| **公共通信** | 骨干网络、5G 核心网 |
| **能源** | 电网、油气管网 |
| **交通** | 航空调度、铁路信号系统 |
| **金融** | 支付清算系统、交易所 |
| **政务** | 电子政务平台、政务云 |
| **科教** | 科研基础设施、大规模数据中心 |
| **水利/卫生** | 重要水利系统、医院信息系统 |

---

## 关基的特别要求

### 检测评估
- 每年至少进行一次安全检测评估
- 安全保护等级不低于第三级
- 网络安全审查（采购网络产品和服务可能需通过审查）

### 运营者责任
| 责任 | 说明 |
|------|------|
| **设置专门安全管理机构** | 明确安全负责人 |
| **主要负责人** | 单位一把手是安全第一责任人 |
| **安全保护措施** | 与关基同步规划、同步建设、同步使用 |
| **应急演练** | 每年至少一次 |
| **数据存储** | 个人信息和重要数据应在境内存储 |
| **采购审查** | 采购可能影响国家安全的网络产品和服务需网络安全审查 |

### 监管处罚
- 不履行关基保护义务 → 最高 100 万元罚款
- 造成严重后果 → 最高 200 万元罚款
- 直接责任人 → 最高 10 万元罚款

---

## 与等保的关系

```
等保是"基线要求"：所有网络/系统都要做
      │
      ▼
关基保护是在等保基础上的"增强要求"
      │
      ▼
等保三级是底线，部分关基可能要求等保四级
```

---

## 关键时间点

```
2017年   网络安全法确立关基制度
2021年   关基保护条例正式施行
2022年   关基认定工作指南发布
         各行业关基识别工作陆续开展
```

---

## 企业建议

- **做好等保三级测评** — 这是底线
- **识别自身是否在关基范围内** — 如果是，尽早成立专门安全团队
- **建立与监管部门的沟通渠道** — 关基运营者需要定期报送安全情况
- **采购审查意识** — 涉及关基的产品采购要注意合规
