# 隐私保护

> 在数据价值和个人隐私之间找到平衡。

---

## 核心法律框架

### 中国
| 法规 | 施行时间 | 核心要求 |
|------|---------|---------|
| **个人信息保护法（PIPL）** | 2021.11.01 | 知情同意、最小必要、目的限制 |
| **数据安全法** | 2021.09.01 | 数据分类分级、出境安全评估 |
| **网络安全法** | 2017.06.01 | 个人信息收集需用户同意 |

### 国际
- **GDPR（通用数据保护条例）** — 欧盟，全球最强隐私法
- **CCPA（加州消费者隐私法案）** — 美国加州

---

## 个人信息保护法核心原则

| 原则 | 说明 |
|------|------|
| **知情同意** | 收集个人数据前必须告知并获取明确同意 |
| **最小必要** | 只收集实现目的所必需的最少数据 |
| **目的限制** | 不能超出收集时声明的目的使用数据 |
| **数据主体权利** | 用户有权查阅、更正、删除自己的数据 |
| **安全保障** | 采取技术和管理措施保障数据安全 |
| **出境管理** | 重要数据出境需安全评估 |

---

## 技术保护手段

| 技术 | 说明 | 适用场景 |
|------|------|---------|
| **数据脱敏** | 遮蔽/替换敏感字段 | 测试环境、分析展示 |
| **差分隐私** | 统计查询时加噪声 | 数据开放共享 |
| **联邦学习** | 数据不动模型动 | 多方联合建模 |
| **同态加密** | 加密状态下的计算 | 金融/医疗敏感场景 |
| **安全多方计算** | 多方协同计算不泄露各自数据 | 跨机构数据协作 |
