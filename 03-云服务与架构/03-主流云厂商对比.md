# 主流云厂商对比

> 国内重点看阿里云，国际看 AWS，两者都有优势。

---

## 三大公有云对比

| 维度 | 阿里云 | AWS | Azure |
|------|--------|-----|-------|
| **市场份额（中国）** | 第一 (~34%) | — | 第三 |
| **市场份额（全球）** | 第三 (~4%) | 第一 (~32%) | 第二 (~23%) |
| **计费复杂度** | 中等 | 复杂 | 中等 |
| **文档中文** | ✅ 完善 | ❌ 大部分英文 | ❌ 英文为主 |
| **合规（中国）** | ✅ 最全（等保、密评） | ⚠️ 受限 | ⚠️ 受限 |
| **AI/ML 服务** | 通义系列 | Bedrock/SageMaker | Azure AI |

---

## 核心服务对照

| 类别 | 阿里云 | AWS |
|------|--------|-----|
| 计算 | ECS | EC2 |
| 无服务器 | 函数计算 FC | Lambda |
| 容器 | ACK / ASK | EKS / Fargate |
| 对象存储 | OSS | S3 |
| 数据库 | RDS (MySQL/PostgreSQL) | RDS (MySQL/PostgreSQL/Aurora) |
| 负载均衡 | SLB | ALB / NLB |
| CDN | CDN | CloudFront |
| 密钥管理 | KMS | KMS |
| 云安全中心 | 云安全中心 | GuardDuty / Security Hub |
| WAF | Web 应用防火墙 | WAF |
| DDoS 防护 | DDoS 高防 | Shield Advanced |

---

## ⚠️ 选择建议

- **国内业务首选阿里云** — 合规、文档、服务都最成熟
- **出海业务选 AWS** — 全球节点覆盖最广
- **微服务 / .NET 生态** — Azure 对微软生态集成最好
- **不要全栈锁定一个厂商** — 关键数据和服务设计为可迁移
